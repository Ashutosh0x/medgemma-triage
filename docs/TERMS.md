## 2. No Medical Advice
The output provided by this system (including urgency scores, heatmaps, and textual explanations) does **not** constitute medical advice or diagnosis. 
- It is intended solely for **adjunctive use** by qualified healthcare professionals.
- It must **never** be used as the sole basis for patient management decisions.

## 3. Human-in-the-Loop Requirement
By using this system, you agree that:
- Every prediction will be verified by a board-certified radiologist.
- You will not rely on the "Urgent" or "Non-Urgent" labels for critical triage without human overview.

## 4. Data Privacy

- No PHI is sent to external cloud APIs by default (Offline Inference Mode).
